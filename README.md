A calibrated multimodal model using **ResNet-50 + ClinicalBERT** on 612K+ radiographs from 42K+ ICU patients, reaching **AUROC 0.91**. Outputs treatment-specific counterfactual predictions so clinicians can reason about risk under alternative fluid, oxygen, and vasopressor strategies in real time.
